## 🌐 タスク管理UI

### アクセス方法
- **あなた（Mac）**: http://localhost:3030
- **チームメンバー（同じWiFi）**: http://192.168.0.180:3030

### 使い方
1. **タスク追加**: タイトル、優先度、締切、担当者を入力して「追加」
2. **ステータス変更**: 「進行中」「完了」ボタンをクリック
3. **フィルタ**: すべて / 未着手 / 進行中 / 完了で絞り込み
4. **自動更新**: 30秒ごとに最新データ取得

## 📅 3日サイクル運用

### Day 1-3: 実行フェーズ
- 各自のタスクを実行
- 完了したらUIで「完了」ボタン
- ブロッカーがあればSlackで報告

### Day 3終了時: 自動レビュー
**ポリが自動実行**（水曜・土曜・火曜 20:00）:
1. 進捗分析
2. ボトルネック検出
3. 次の3日間タスク提案
4. Slackに報告投稿

### レビュー後: 次サイクル準備
- 報告を確認
- 必要に応じてタスク追加・修正
- 優先度を再調整
